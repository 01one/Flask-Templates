About Dynamic URLs in Flask

This tutorial demonstrates how Flask handles dynamic URLs and routes with parameters. Flask provides a powerful routing system that allows you to create flexible URL patterns.

Flask Route Converters

Converter Description Example
string Default, accepts any text without slashes /user/<username>
int Accepts integers /post/<int:post_id>
float Accepts floating-point values /product/<float:price>
path Like string but also accepts slashes /file/<path:filename>
uuid Accepts UUID strings /user/<uuid:id>

URL Building with url_for()

Flask's url_for() function helps you generate URLs for routes, making your application more maintainable:

# Instead of hardcoding URLs like this:
<a href="/user/user1">user1's Profile</a>

# Use url_for() like this:
<a href="{{ '{{' }} url_for('profile', username='user1') {{ '}}' }}">User1's Profile</a>

This ensures that if you change your route patterns, all your links will automatically update.
